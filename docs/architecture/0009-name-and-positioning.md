# ADR 0009: Name and positioning

## Recommended repo name
Waywarden

## Positioning
Waywarden is a public, slim, extensible agent harness with profile packs for executive assistance, coding, home automation, and future domains.

## Why
- public-friendly
- not tied to a single persona
- works for a reusable core harness
- implies guidance and staying on course
- distinct from existing OpenHarness repos

## Rejected
- anything with OpenHarness
- anything too tied to smart home
- anything too tied to coding only
- anything too cute
