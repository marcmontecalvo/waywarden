# Roadmap Research: badlogic/babbletui

## What it is

A minimal TUI library for chat interfaces.

## Why it should be tracked

It is a relevant reference if WayWarden later wants a local terminal UI or lightweight operator console.

## Why it is not a “now” item

Current value is architectural inspiration only. It should not drive current repo structure unless WayWarden explicitly commits to a TUI surface.

## Possible future uses

- local operator console
- debugging / playback UI
- agent status, logs, and intervention controls in terminal form
