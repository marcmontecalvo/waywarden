# Security Policy

## Supported Versions

Currently, Waywarden is in active initial development (v0.x). Security patches will be applied to the primary `main` branch. Older versions are not actively maintained with security patches at this time.

## Reporting a Vulnerability

If you discover a security vulnerability within Waywarden, please do NOT report it by creating a public GitHub issue.

Instead, please send an ethical disclosure via email to the maintainers (or use GitHub Security Advisories privately).

We will try to acknowledge your report promptly and provide an estimated timeline for resolution and disclosure.
